## 🔍 Feature Highlights

### 🎯 Gesture Detection
- **Event-based** (back button listener, no polling)
- **Smart debouncing** (600ms window, 30s cooldown)
- **Haptic feedback** (vibration on each tap)
- **Optional accelerometer** (supplementary detection)

### ⏱️ SOS Countdown
- **5-second silent countdown** (no audio)
- **Animated overlay** (smooth, non-blocking)
- **Per-second vibration** (haptic feedback)
- **Color intensity** (increases as timer expires)

### 🛡️ Cancellation Methods
- **PIN verification** (encrypted, 4-6 digits)
- **Long-press detection** (2-second hold required)
- **Cannot cancel after 1s** (forced completion)
- **User-friendly** (multiple options)

### 📍 Location Service
- **High-accuracy GPS** (up to 5m accuracy)
- **Intelligent timeout** (30s max wait)
- **Fallback caching** (uses last known if GPS fails)
- **Reverse geocoding** (gets address)

### 📱 Contact Notification
- **Primary API** (JSON delivery)
- **SMS fallback** (if network unavailable)
- **Automatic retry** (queued persistence)
- **Formatted message** (includes maps link)
