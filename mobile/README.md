# GuardianX (mobile)

Expo prototype for the GuardianX personal safety app.

Quick start:

1. Install dependencies:

```bash
cd mobile
npm install
```

2. Add logo: place the provided GuardianX logo at `mobile/assets/guardianx-logo.png`.

3. Run:

```bash
npm start
```

Notes:
- Google Maps, Directions, Places and MediaPipe integrations are simulated/placeholders.
- Backend endpoints referenced in `src/services/api.js` should point to the existing `Backend/` server.
